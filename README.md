## Описание

В ходе выполнения лабораторной работы были изучены основы работы с DOM-деревом, обработка событий JavaScript, а также создание динамического интерфейса. Было разработано два приложения:

1. **TODO-лист** - классическое приложение для управления задачами с возможностью добавления, отметки выполнения и удаления задач.
2. **Таймер** - приложение для отсчёта времени с кнопками управления.

В процессе работы были освоены:

- Поиск и создание элементов DOM
- Обработка событий (клик, ввод с клавиатуры)
- Работа с формами
- Динамическое обновление интерфейса
